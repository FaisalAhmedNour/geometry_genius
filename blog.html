1. What is DOM and what is the purpose of DOM?

DOM means Document Object Model. It is a document represented as object. It represents the full html page in an object. With it program can add, remove or change the elements, attributes, styles, text and everything in the document.

2. How will you select HTML elements using DOM? Name the DOM methods.

I can select Html elements by many ways like using tag name, class, id etc. Example: document.getElementsByClassName(‘class’);

DOM methods:

  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

3. What is event bubble?

Event bubble is a concept in DOM, it happens when any event occurs in an element. First it goes to the innermost element in that element. Then starts climbing up to its parents until the uppermost element.

4. What is a callback function and why will you use it?

Call back function Is a function which is used an argument in another function. By calling it that function completes it actions.
I will use call back function to organize my code better and specially for control the sequence of operations. I can also use It for avoiding the repetition of writing the same code.
